# 免费版全球资金流向覆盖矩阵

这份矩阵的目标不是追求“所有市场都有精确净流入金额”，而是用免费数据构建一个可长期运行的全球资金偏好监控系统。

## 三层信号体系

### 1. 真实流量

能直接反映净流入或申赎的数据，例如：

- A股北向资金
- 美股现货比特币 ETF 净流入
- 部分 ETF 公开申赎

特点：

- 解释力最强
- 免费覆盖面有限
- 多数只能覆盖少数市场

### 2. 仓位变化

能反映机构持仓方向变化的数据，例如：

- CFTC COT 期货持仓周报
- 13F 持仓
- 稳定币供应量变化

特点：

- 适合做周频或专题增强
- 对日报有价值，但不是所有资产都有

### 3. 代理流向

用价格、相对强弱、成交量、波动率、利率、信用利差等指标推断资金偏好。

特点：

- 免费覆盖面最广
- 最适合日频 GitHub Actions 自动化
- 是当前项目的主引擎

## 当前不建议强行纳入日报主结论的板块

- 俄罗斯股市
- CDS 与互换
- 全球 OTC 外汇真实资金流
- PE / VC / 对冲基金实时仓位
- 各国信用债最细分结构化品种

这些板块不是完全不能做，而是免费条件下难以长期稳定、低维护地日更。

## 推荐的产品分层

### 日报主层

使用免费代理信号输出：

- 全球区域股市强弱
- 全球行业轮动
- 美债与信用风险偏好
- 外汇方向
- 商品方向
- 加密市场风险偏好

### 周报增强层

后续可接入：

- CFTC COT 周报
- A股北向资金
- BTC ETF 净流入
- 稳定币供应变化

这样可以把“代理流向”与“真实流量补充验证”结合起来。

## 对当前项目的定义

当前项目最准确的产品定义是：

`免费版全球大类资产资金偏好与轮动早报`

而不是：

`全球所有市场真实净流入净流出监控`

前者在免费条件下可长期运行、解释力强、适合自动化；后者在免费条件下很难完整成立。
